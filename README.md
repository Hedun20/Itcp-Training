# ITCP Training

ITCP Training is the full-stack learning platform for ITCP Europe. It contains a branded learner experience, an administration CMS, backend-scored assessments, persistent progress, and secure role-based authentication.

The application is organized as an npm workspace with a React/Vite client in `client/` and a TypeScript/Express/MongoDB API in `server/`.

Detailed setup, architecture, API, security, seeding, and deployment guidance is added under `docs/` as part of the implementation.
